Tree Map

Examining the hierarchy of sample news coverage categories of The New York Times

Tree Map

This tree map utilizes the same untouched data from the cluster chart, as it analyzes the visual hierarchy of sample news categories and subcategories in a node-based parent/child relationship. A legend was omitted, as the color hue is primarily used to help display the level in the hierarchy

Tree Map Uses

The tree map displays a visual hierarchy through the use of nested rectangles in a tree-like structure. Tree maps work well with hierarchies that deal with quantitative values attached to the categories to showcase the division between the categories. They work well in providing quick top-level overviews of the relationship, but can get cluttered with large datasets. They work well in displaying how parts form the sum, such as a tree map depicting how resources are consumed in a country.

Marks and Channels

  • Marks: Rectangle, Containment (implied)
  • Channels: Color Hue, Area(2D size)
